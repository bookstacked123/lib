The Filename Odyssey

Theme: “How hard can filenames be?”
Spoiler: Harder than expected.

Work Completed:
- Began the day with a simple goal: evolve our existing filename conventions into a new, “future‑proof” structure.
- The target format:
`audio.tyler_mcgill_unknownsoldier_bg.2026y_03m_01d_18h_07m.0.[avenmore,machine_music].mm8d1dq1t88yuukssh.mp3`
- Immediately realized this filename has the energy of a passport number, a timestamp, a metadata sandwich, and a cryptographic sneeze all rolled into one.
- Broke down the structure into components:
  - Media type (audio)
  - Author (tyler_mcgill)
  - Title (unknownsoldier_bg)
  - Timestamp (2026y_03m_01d_18h_07m)
  - Version (0)
  - Tags ([avenmore,machine_music])
  - Entropy string (mm8d1dq1t88yuukssh)
  - Extension (mp3)
- Debated whether the entropy string should be random, hash‑based, or generated by sacrificing a USB stick to the tech gods.
- Settled on: random but deterministic‑seed‑friendly, because chaos is fun but reproducibility is nicer.
- Documented the rules for each segment, including:
  - Lowercase normalization
  - Underscore vs dot boundaries
  - Sanitization rules
  - Maximum filename length (255 chars, thanks to the filesystem police)
- Created several prototype filenames, including one that accidentally exceeded 300 characters and one that contained an emoji because I forgot to strip Unicode.
- Tested the structure with various media types and discovered that video.tyler_mcgill_... looks intimidating enough to scare small children.

Successes:
- Finalized the full filename template with clear rules.
- Ensured the structure is human‑readable, machine‑parsable, and mildly terrifying (in a good way).
- Confirmed that the entropy block makes filenames look like they were generated by a cyberpunk librarian.

Failures:
- First attempt at tag formatting produced:
`[avenmore,,machine_music,]`
which looked like a CSV file having a panic attack.
- Accidentally generated a filename with two timestamps because I forgot I already added one.
- Briefly considered switching careers to pottery.

End‑of‑Day Mood:  
Proud. Slightly dizzy. Filename‑shaped hallucinations when blinking.

End-of-day due date. 2026-02-23.
